# 🎓 Parcial 2 - Software

## 📝 Descripción
Página web estática servida mediante Nginx. Este proyecto fue desarrollado como evaluación parcial, con el objetivo de demostrar un flujo de trabajo real versionando un proyecto con **Git**, **GitHub** y su contenedorización con **Docker**.

**Alumno:** Juan María Elsener  
**Fecha:** 24 de Junio de 2026

## 🛠️ Tecnologías Utilizadas
- **HTML5** — Estructura de la página.
- **Nginx (Alpine)** — Servidor web ligero para servir los archivos estáticos dentro del contenedor.
- **Docker** — Contenedorización de la aplicación.
- **Git & GitHub** — Control de versiones, manejo de ramas (branching/merge) y repositorio remoto.

## 📋 Requisitos Previos
Antes de ejecutar este proyecto necesitás tener instalado:
- [Git](https://git-scm.com/) (versión 2.x o superior)
- [Docker Desktop](https://www.docker.com/products/docker-desktop/) (activo y ejecutándose)
